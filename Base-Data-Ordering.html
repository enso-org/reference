Ordering
Types representing the ordering of values.
These are intended to be returned from the compare_to function, that has a type as follows for a type A: A.compare_to : A -> Ordering.
The result should be returned in terms of how this orders in comparison to that. So, if this is greater than that, you should return Greater.
